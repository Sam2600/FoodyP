1. How to start the server ?
   - python manage.py runserver

2. How to reset the migration and database ?
   - Delete the db.sqlite3 File
   - cd to the your application directory -> (store)
   - 1.1 rm -f store/migrations/0*
   - 1.2 python manage.py migrate store zero
   - 1.3 python manage.py makemigrations
   - 1.4 python manage.py migrate

3. How to create admin user account ?
   - python manage.py createsuperuser
